# Azure Security Labs — Microsoft Cybershujaa Program

A collection of hands-on Azure security labs completed as part of the 
Microsoft Cybershujaa cybersecurity training program.

These labs cover core Azure security services aligned with the 
AZ-500 and SC-200 certification tracks.

---

## Labs Completed

| Lab | Topic | Key Services |
|-----|-------|-------------|
| Lab 07 | Azure Key Vault & Always Encrypted | Key Vault, Azure SQL, Entra ID |
| Lab 08 | Log Analytics & Data Collection | Azure Monitor, Log Analytics, DCR |
| Lab 09 | Microsoft Defender for Cloud | Defender for Servers Plan 2, CWPP |
| Lab 10 | Just-in-Time VM Access | JIT, NSG, Defender for Cloud |
| Lab 11 | Microsoft Sentinel | SIEM, SOAR, Logic Apps, KQL |

---

## Skills Demonstrated

- Azure identity and access management (RBAC, Access Policies)
- Secret and key management using Azure Key Vault
- Database column encryption using Always Encrypted
- Centralised log collection using Azure Monitor Agent and DCRs
- Threat protection using Microsoft Defender for Cloud
- Attack surface reduction using Just-in-Time VM access
- SIEM implementation and incident response using Microsoft Sentinel
- Security automation using Logic Apps playbooks
- KQL (Kusto Query Language) for threat hunting
